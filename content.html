This is a tab bar powered by Alpine.js

Tab bars are used to break information up into distinct sections which are too succinct to warrant their own pages. If one of the sections were to get too large, it should go on its own page to avoid cramming a small container full of words.

All it takes to use a JS library is a single line

Thanks to the magic of Skypack and a little Regex, all you have to do to import and use a JavaScript library is import Alpine from 'alpinejs'. Skypack send preautomized packages to everyone who visits your site. Note that JS libraries used this way won't work in IE11.

Alpine makes it easy to sprinkle in interactivity

AlpineJS is "A rugged, minimal framework for composing JavaScript behavior in your markup". Once you use a library in one component, you can use it from other components without any additional overhead (since it's cached from then on).
